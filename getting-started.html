Download source

The latest version of eXo UI Guidelines project can be downloaded directly from GitHub.

Download eXo UX Guidelines

After downloading, you can use code editors, such as Notepad++, Text Editor, Quanta Plus or Sublime Text, to modify files. Also, you are required to have some knowledge of CSS and HTML.

File structure

The structure of eXo UX Guidelines is similar with that of Bootstrap.

After downloading, you need to unzip the compressed folder. The structure of eXo UX Guidelines is displayed below:

  exo-ux-guidelines /
  ├── css/
  │   ├── bootstrap.css
  │   ├── bootstrap.min.css
  │   ├── exo-platform.css
  ├── js/
  │   ├── bootstrap.js
  │   ├── bootstrap.min.js
  ├── img/
  │   ├── icon_start.png
  │   ├── icon_pattern.png
  │   ├── exo-platform-icons
  └── README.md

Most CSS and JavaScript files are from Bootstrap. In addition, to build the eXo Platform style, exo-platform.css and uxp-pattern.css are customized by eXo Platform Team.

Please note that most JavaScript plugins require jQuery to be included. Besides, some free JavaScripts are also applied into eXo UX Guidelines to create dynamic samples.

HTML template

eXo UX Guidelines uses a basic HTML template provided by Bootstrap.

Here is a typical HTML file:

  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>Bootstrap Template</title>
  5.   </head>
  6.   <body>
  7.     <h1>Hello, world!</h1>
  8.     <script src="http://code.jquery.com/jquery-latest.js"></script>
  9.   </body>
  10. </html>

To make this a Bootstrapped template with the eXo Platform style, just include the appropriate CSS and JavaScript files of Bootstrap and the customized CSS that is provided by eXo Platform Team.

Please note that exo-platform.css is the Core.css file that is located in skin/css of eXoResource.war in the eXo Platform package. You can copy, rename and put it in the general css folder that contains Bootstrap CSS. 

  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>Bootstrap Template</title>
  5.     <!-- Bootstrap -->
  6.     <link href="css/bootstrap.css" rel="stylesheet">
    <!-- eXo Platform CSS -->
  7.     <link href="css/exo-platform.css" rel="stylesheet">
  8.   </head>
  9.   <body>
  10.     <h1>Hello, world!</h1>
  11.     <script src="http://code.jquery.com/jquery-latest.js"></script>
  12.     <script src="js/bootstrap.min.js"></script>
  13.   </body>
  14. </html>
