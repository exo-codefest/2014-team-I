Overview

Tab is a resource you can use in the same browser to open up another content. The content of a page, which is grouped in different tabs, can be viewed easily.

Switching among tabs helps users to browse more content flexibly without scrolling down or going to another pages.

When to use?

Tabs should be used in cases:
The content of page or popup is so long.
There are more 2 sections.
The content of each tab can be viewed separately from each other.

Tabs are often put on a horizontal or vertical bar of the content area as buttons or links. The selected tab has its own visual effect that is different from the others. The content of each tab is displayed in the same area with a similar structure.

Status

Status Description
Default status The tab can be clicked to view its content. There is a CSS effect when it is hovered over.
Being viewed status The tab is being viewed, and its content is displayed. There is no or a minor CSS effect (for example, its label in lighter color) when the tab is hovered over.

Rules

TR-01 Different tabs have the same CSS style.
TR-02 The tab content is displayed in the same area.
TR-03 The tab title is limited to 20 characters.
TR-04 By default, there is only one tab being viewed at once.
TR-05 No page load when a tab is clicked.
